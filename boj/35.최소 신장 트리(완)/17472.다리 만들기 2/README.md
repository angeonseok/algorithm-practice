# 17472.다리 만들기 2

## 문제 설명
섬을 번호로 구분하고, 만들 수 있는 다리를 간선으로 변환한 뒤 모든 섬을 최소 비용으로 연결하는 문제입니다.

## 풀이 파일

- `17472.다리 만들기 2(kruskal).py`: 가능한 다리를 간선 목록으로 만든 뒤 Kruskal 알고리즘으로 MST를 구성합니다.
- `17472.다리 만들기 2(prim).py`: 가능한 다리를 인접 정보로 만든 뒤 Prim 알고리즘으로 MST를 구성합니다.

## 풀이 정리

### 핵심 아이디어

같은 문제를 MST 관점에서 풀되, 최소 신장 트리를 구성하는 알고리즘만 다르게 비교합니다.
