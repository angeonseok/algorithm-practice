# 26009.곱의 합

## 문제 설명
세 정수 a,b,c 가 주어졌을 때, 를 998244353 으로 나눈 나머지를 출력하라.

## 입력
첫 번째 줄에 테스트 케이스의 수 TC가 주어진다.
이후 TC개의 테스트 케이스가 새 줄로 구분되어 주어진다.
각 테스트 케이스는 다음과 같이 구성되었다.
- 첫 번째 줄에 세 정수 a, b, c 가 주어진다. (1 ≤ a, b, c ≤ 109)

## 출력
각 테스트 케이스 마다 한 줄씩, 정답을 출력하라.
